CMPSC 457 Lab #5

By: Matthew Panetta

Instructions

In this lab I needed to:
  • Draw a 3D ragdoll
  • Draw each individual part independantly
  • Provide a dancing animation
  • Allow freeform rotation using arrow keys

My Implementation

  • I made each part seperately. The head is a sphere, while everything else is a cylinder.
  • When 'a' is pressed, the ragdoll will perform a funky fresh dance move.
  • While not animating, the ragdoll can be freeform rotated using the arrow keys.
  • Rotations are fixed between two intervals, so the ragdoll won't break.
  • Up = Left Leg, Down = Right Leg, Left = Left Arm, Right = Right Arm
  • The robot has a green shirt and jeans on.
